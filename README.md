# Affordable Luxury Japan Trip Microsite

Three-page informational landing microsite for an affordable luxury Japan travel planning concept.

## Pages

- `index.html` — main affordable luxury Japan trip guide
- `couples.html` — Japan honeymoon/couple trip guide
- `family.html` — Japan family trip guide
- `styles.css` — shared design system

## Design system

Generated with UI/UX Pro Max query:

`travel agency Japan trip affordable luxury itinerary premium family honeymoon India informational guide`

Direction:
- Storytelling-driven information funnel
- Premium editorial travel look
- Warm luxury colors: ink, stone, paper, gold, restrained Japan red
- Bodoni Moda headings + Jost body
- Mobile-first, accessible, reduced-motion support
